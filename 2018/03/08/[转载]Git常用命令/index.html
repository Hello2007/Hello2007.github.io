本人曾遇到的问题

  1. Q:fatal: remote origin already exists.
    A:git remote -v查看一下，然后git remote rm origin删除已存在的远程库remote，重新git remote add....或者git remote add [换个名字]....
  2. Q:fatal: refusing to merge unrelated histories
    A:git pull origin master ----allow-unrelated-histories
